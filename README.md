# Azin.Shadi

با سلام خدمت همه دوستان عزیز
خب قراره با هم یه پروژه کوچیک بزنیم و کنار هم یه چیزایی یاد بگیریم مشکلات و ضعف های خودمون رو پیدا کنیم و باهم برطرفش کنیم
---------------------------------------------------------------------------------------------------------------------------
پروژه مون تقریبا یه پروژه مدیریت یه فروشگاه هست که میتونیم محصول اضافه منیم پاک کنیم خروج و ورود محصول و ... یه سری کارا رو انجام بدیم که شرح کاملشو میگم خدمتتون

از اونجایی که خانوم مریم احمدی آخر سی شارپ دوره سی شارپ پیشرفته هستن قراره کلاسا و پراپرتی ها و یه سری متد هارو پیاده سازی کنن

و زحمت پیاده سازی دیتابیس و یه سری کارای رو اقای جمشیدی و خسرو زاده قراره انجام بدن

امیدوارم کنار هم کلی چیز یاد بگیریم و این فایل زود به زود بروزرسانی میشه و مشخص میکنم هر فرد کدوم از قسمت پروژه رو باید زحمتشو بکشه
